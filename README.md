# 3-Style Trainer

> A web tool to practice 3x3 blindfolded (BLD) solves in Edge, Corner, or All modes.

---

## 🔹 Demo

Available on GitHub Pages:  
`https://yougo-rgb.github.io/3style-trainer/`

---

## 🧩 Features

- Mode selection:
  - **Edge**
  - **Corner**
  - **All**
- Dynamic cube display using **Cubing.js**
- Random scramble generation
- Buffer selection

---

## ⚙️ Installation / Local Usage

1. Clone the repository:

```bash
git clone https://github.com/Yougo-rgb/3style-trainer.git
cd 3style-trainer
````

2. Open `index.html`

> A local server is optional but recommended for some browsers, e.g., VSCode Live Server
